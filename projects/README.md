# Projects

This folder contains all **mini-projects and end-to-end projects** completed as part of the **LogicMojo AI & Machine Learning (Jan 2026)** program.

## What goes here
- Mini projects built during the course
- Concept-to-implementation projects
- Problem-solving projects applying Python, Data Structures, and OOP
- Larger AI / ML projects added later in the program

## Structure
Each project will have its **own folder** with:
- Source code or notebooks
- Data (if applicable)
- A dedicated `README.md` explaining the problem, approach, and outcome

## Purpose
These projects demonstrate:
- Practical application of concepts
- Clean code organization
- Problem-solving and implementation skills
