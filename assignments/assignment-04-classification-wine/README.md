# Wine Classification

## Project Overview
This project focuses on classifying wine samples into different categories based on their chemical properties.

It is a **supervised machine learning classification problem** where the goal is to predict the wine class using features such as alcohol, malic acid, magnesium, flavanoids, color intensity, and others.

---

## Dataset Information
- **Dataset Name:** Wine Dataset
- **Source:** Scikit-learn
- **Link:** https://scikit-learn.org/stable/modules/generated/sklearn.datasets.load_wine.html

### Target Variable
- `target`
  - 0 → Wine Class 0
  - 1 → Wine Class 1
  - 2 → Wine Class 2

### Features
The dataset contains multiple chemical analysis features, including:
- alcohol
- malic_acid
- ash
- alcalinity_of_ash
- magnesium
- total_phenols
- flavanoids
- nonflavanoid_phenols
- proanthocyanins
- color_intensity
- hue
- od280/od315_of_diluted_wines
- proline

---

## Objective
The main objective of this project is to:
- understand the dataset
- perform exploratory data analysis (EDA)
- clean and preprocess the data
- train classification models
- evaluate model performance
